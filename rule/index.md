# 利用規約及びプライバシーポリシー

## 利用規約

本規約は、さめすきー(以下「当インスタンス」)の利用条件を定めるものです。ユーザーの皆様には、本規約に従って当インスタンスをご利用いただきます。

### 1. 禁止事項

ユーザーは、当インスタンスの利用にあたり、以下の行為を行ってはなりません。

- **法令または公序良俗に違反する行為**

- **犯罪行為に関連する行為**

- **他のユーザー、第三者の知的財産権、プライバシー権、名誉権を侵害する行為**

- **サーバーまたはネットワークの機能を破壊したり、妨害したりする行為**

- **当インスタンスの運営を妨害するおそれのある行為**

- **大量のリクエストを短時間に送り、負荷をかける行為（DoS攻撃など）**

- **その他、管理者が不適切と判断する行為**

### 2. アカウント管理

- **ユーザーは、自己の責任においてアカウントを管理するものとします。**

- **管理者は、ユーザーが規約に違反した場合、事前の通知なくアカウントの停止または削除を行うことができるものとします。**

### 3. 免責事項

- **管理者は、当インスタンスの利用により生じた損害について、一切の責任を負いません。**

- **当インスタンスは、メンテナンスや不測の事態により停止することがあります。データの保存を保証するものではありません。**

### 4. 規約の変更

管理者は、必要と判断した場合には、ユーザーに通知することなくいつでも本規約を変更することができるものとします。

## プライバシーポリシー

当インスタンスは、ユーザーのプライバシー情報の取扱いについて、以下のとおり方針を定めます。

### 1. 収集する情報

当インスタンスでは、以下の情報を収集・保持します。

- **IPアドレス（必須）** : サーバーの不正アクセス防止、スパム対策、および法令遵守のためにすべてのアクセスにおいてロギングを行っています。

- **メールアドレス（任意）** : パスワードのリセットや通知の送付を希望する場合にのみ、ユーザーの意志で登録いただけます。

### 2. 利用目的

収集した情報は、以下の目的でのみ利用します。

- **サービス提供の維持およびセキュリティ確保**

- **パスワード再発行などの本人確認（メールアドレス登録時）**

- **法令に基づく正当な開示請求への対応**

### 3. 第三者への提供

当インスタンスは、法令に基づく場合を除き、収集した個人情報を第三者に提供することはありません。

### 4. 外部アナリティクス・広告について

当インスタンスでは、Google Analyticsなどの外部アクセス解析ツールや、第三者配信の広告ネットワークは使用していません。

### 5. データの削除

ユーザーは、自身のアカウントを削除することで、登録された情報を削除することができます。  
ただし、バックアップデータやサーバーログ内に残る記録については、一定期間の経過後に自動的に消去されます。
